# F221317_vaegan_mnist_project_prep
VAE, GAN and VAEGAN architecture used as proof of concept for a proposed specialised VAEGAN for aerofoils. The code for the VAE and GAN was adapted from Keras' publicly available models.

Edit (12/05/26): A GCPI-VAEGAN model has been added. This was built upon the existing models shown within this collection. 
                 Key changes include:
                 - Physics Predictor
                 - Geometry Constraints
                 - Multi-stage training
                 - Hyperparameter changes
